Application Patterns


Layouts

Home Page

This template accommodates a mix of messages from the business and featured news and events to orient users to the site and provide a glimpse into the most recent site content. Primary areas:

  • A hero carousel displays introductory statements or feature stories
  • A "section teaser carousel" orients visitors to key site sections
  • A bottom zone with dynamic content modules displays a rotating assortment of news, events, and promotional content

Content Index Page

This template contains a large area for featured content and a uniform grid of fixed size index link modules that provide an overview of child pages of the represented section.


Jump Navigation

This template allows users to move quickly through very long pages. Jump subnavigation bar sticks to top of page, beneath main navigation.


Full Width Content

This template addresses the need for full page width images and text. It should be used selectively, when full width graphics or other full width components need to be displayed.


Blog Post: Main Content with Right Rail

This template accommodates a large image (as needed) and space for long-form text and other elements in the page body. The right rail is reserved for associated content. The template does not prescribe or recommend components for the right rail, but indicates the types that may be appropriate for different associated content requirements. This template provides a foundation for blog posts or articles but may be modified as needed.


Lightbox View Page

This template should be used to display content that requires very large images with captions. It contains a large area with a uniform grid of fixed size view link modules. Each module includes a thumbnail image, a link to another page (if needed), and a link to launch a lightbox view. The lightbox carousel can be used in 2 ways:

  • to display larger versions of each thumbnail image on the page body in the same order
  • to display a set of images unique to the view link module from which it is launched
