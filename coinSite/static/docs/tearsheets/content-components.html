Dynamic Content Module

Used on the home page to feature a selection of varied content from deeper within the site

Elements:

  • May feature news, event, or story
  • Image is optional
  • Text elements can include headline, date, location, introductory statement, etc.
  • "Read more" links to content page for given item
  • Associated tags are displayed at bottom of module
  • Tags link to topic pages featuring a selection of content related to given tag
  • Height adjusts to fit display content
  • Several modules may feature on a home page. 6-9 recommended.

View Link Module

Provides thumbnail view of a larger image viewable in lightbox, utilizing title, image, description, and link elements

Usage Notes:

  • Second link launches the lightbox
  • As with the Index Link Module, the uniform class on the top-level content div ensures that they will be all be the same height, determined by the module containing the most text.
  • When lightbox launches, it displays the module's thumbnail image, with others accessible through next/previous controls
  • Default behavior: the lightbox contains a carousel that displays the images from the other index link modules on the view index page
  • Alternate behavior: the lightbox contains a carousel displaying a set of images specific to just given the index link module

Associated Templates:

  • Views Index
